## 项目简介

这是一个个人作品集网站，展示我的开发工作和技术能力。

## 主要特性

- 🌙 **暗色/亮色模式** - 自定义主题切换
- 🌐 **中英文切换** - 国际化支持
- ✨ **交互动效** - 丰富的动画和特效
- 📱 **响应式设计** - 适配各种设备
- 🎨 **手绘装饰** - 独特的视觉风格

## 技术栈

使用了 Next.js 16 App Router、TypeScript、Tailwind CSS 和 Framer Motion 构建。

## 开发心得

这个项目让我深入探索了现代前端的最佳实践，包括 CSS 动画、性能优化、组件设计等方面。
