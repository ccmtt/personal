## 项目简介

一个 Git 提交历史可视化工具。

## 主要特性

- 🔀 **分支可视化** - 直观展示分支和合并
- 📊 **提交图表** - D3.js 绘制的交互图表
- 🔍 **历史追溯** - 快速定位关键提交

## 技术栈

Vue.js, D3.js, Git API
